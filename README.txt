Tatai

Requirements
-Your system must have FFMPEG installed. To install FFMPEG:
	1. Open a terminal by typing Ctrl+Alt+T
	2. Type the following command into the terminal:
		sudo apt install ffmpeg
	3. Wait for FFMPEG to finish installing
-Your system must have Open JFX installed. To install Open JavaFX, type the following command into the terminal:
	sudo apt install openjfx
-Your system must have HTK installed, and the HTK configuration file must be your Documents
	folder
-The executable file Tatai.jar must be in the same directory as the sound file pass.mp3

Getting Started
-Connect a microphone to your computer and set it as the default input device
-Run the executable file Tatai.jar by the following process:
	1. Open a terminal
	2. Navigate to directory containing the Tatai.jar and pass.mp3 files
	3. Type the following command into the terminal:
		java -jar Tatai.jar
